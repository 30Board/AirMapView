AirMapView

A library to enable map user interfaces for device with and without Google Play Services.

Get Started

Multiple Providers

Supports multiple native map providers to provide the best map experience for different device types such as Google Nexus vs Amazon Fire.

All the Maps

Swap between supported providers at runtime to provide the best experience based on context.

Maps for Everyone

AirMapView will fallback to a web based map implementation for devices that do not support any native map provider.
